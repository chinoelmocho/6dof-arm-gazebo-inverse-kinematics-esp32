# Capturas y fotos del proyecto

Pon aquí las imágenes que quieras mostrar en el README principal (fotos del
brazo real, capturas de Gazebo/RViz, gifs de la trayectoria, etc.).

Sugerencia de nombres (el README principal ya los referencia con estos
nombres; si usas otros, actualiza los enlaces en `README.md`):

- `gazebo.png` — el brazo simulado en Gazebo.
- `rviz.png` — visualización en RViz.
- `arm-real.jpg` — foto del brazo físico con el ESP32.
- `demo.gif` — trayectoria en movimiento (simulada o real).
